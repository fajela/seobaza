ВАЖЛИВО: Додайте PDF файл "Указівки для асесорів"

Завантажте файл з https://seobaza.com.ua/quality-raters-guidelines-short-ukrainian.pdf
та збережіть його як:

C:\seobaza\seobaza-new\public\quality-raters-guidelines-short-ukrainian.pdf

Альтернативно, якщо у вас є локальна копія:
- Знайдіть файл quality-raters-guidelines-short-ukrainian.pdf
- Скопіюйте його в папку public/
- Переконайтесь що назва файлу точна: quality-raters-guidelines-short-ukrainian.pdf

Після додавання файлу, він буде доступний за посиланням:
http://localhost:3000/quality-raters-guidelines-short-ukrainian.pdf

І в розділі "База знань":
http://localhost:3000/knowledge-base
