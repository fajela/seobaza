# OpenGraph зображення для SEO BAZA

## Потрібне зображення: `og-image.png`

Розмір: **1200 x 630 пікселів**

### Що має бути на зображенні:

- Логотип SEO BAZA
- Текст: "SEO BAZA - українська SEO-спільнота"
- Фон у стилі сайту (можна використати градієнт або однотонний)
- Акцентні кольори з сайту (#7CFF3D зелений, #0052CC синій)

### Як створити:

**Варіант 1: Canva (найпростіше)**
1. Перейди на https://www.canva.com
2. Створи Custom Size: 1200 x 630 px
3. Додай лого та текст
4. Експортуй як PNG

**Варіант 2: Figma**
1. Створи фрейм 1200 x 630 px
2. Додай елементи дизайну
3. Експортуй як PNG

**Варіант 3: Photoshop/GIMP**
1. Новий файл: 1200 x 630 px
2. Додай дизайн
3. Save as PNG

### Після створення:

1. Назви файл `og-image.png`
2. Помісти в папку `public/`
3. Закоміть і запуш на GitHub
4. Vercel автоматично задеплоїть

### Перевірка:

Після деплою перевір OpenGraph теги:
- https://www.opengraph.xyz/
- https://www.linkedin.com/post-inspector/
- Facebook Debugger: https://developers.facebook.com/tools/debug/

### Поточний статус:

❌ Зображення `og-image.png` ще не створене
✅ Metadata налаштована для всіх сторінок (статті, події, тести)
✅ Twitter Cards налаштовані
✅ Canonical URLs додані

Коли створиш зображення - просто помісти його сюди і запуш!
